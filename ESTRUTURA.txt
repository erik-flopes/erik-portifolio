⚙️ CONFIGURAÇÃO FLASK
─────────────────────

app = Flask(__name__, 
    static_folder='static',           # Arquivos CSS, JS, imagens
    static_url_path='/static',        # URL base para arquivos
    template_folder='templates')      # Pasta com HTML

Rotas:
  GET  /                → Serve template/index.html
  GET  /static/css/* → Serve CSS files
  GET  /static/js/*  → Serve JS files
  GET  /api/projects → Retorna projetos (JSON)
  GET  /api/skills   → Retorna skills (JSON)
  GET  /api/experience → Retorna experiência (JSON)
  GET  /api/stats    → Retorna estatísticas (JSON)
  POST /api/contact  → Recebe mensagem (JSON)
