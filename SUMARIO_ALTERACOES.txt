   ├── Conteúdo: Todo CSS do projeto

   ├── Tamanho: ~500 linhas
   ├── Conteúdo: Toda lógica JavaScript
   ├── Mock API para testes (projeto funciona sem backend)
   ├── Fetch dinamico de dados da API
   ├── Filtros de projetos
   ├── Validação e envio de formulário
   ├── Animações Intersection Observer
   └── Toast notifications

3. templates/index.html
   ├── HTML limpo e semântico
   ├── Referências dinâmicas via url_for() Flask
   ├── Link para CSS: {{ url_for('static', filename='css/style.css') }}
   ├── Script para JS: {{ url_for('static', filename='js/script.js') }}
   ├── Sem inline CSS ou JavaScript
   └── Meta tags e fonts externas mantidas

═══════════════════════════════════════════════════════════════════════════

🔄 FLUXO DE FUNCIONAMENTO
────────────────────────────

1. Usuário acessa http://localhost:5000
   ↓
2. Flask serve templates/index.html
   ↓
3. index.html carrega static/css/style.css
   ↓
4. index.html carrega static/js/script.js
   ↓
5. script.js faz fetch de:
   - /api/stats
   - /api/projects
   - /api/skills
   - /api/experience
   - /api/contact (POST)
   ↓
6. app.py retorna dados JSON
   ↓
7. script.js renderiza dinamicamente

📝 NOTAS IMPORTANTES

  - Funciona completamente sem o backend rodando
  - Permite testar frontend isoladamente

  - Os dados reais da API são usados
  - Mock API é sobrescrita pelo fetch real
