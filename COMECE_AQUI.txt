🛠️ COMO EDITAR O PROJETO
──────────────────────────

EDITAR ESTILOS (CSS):
  → Modifique static/css/style.css
  → Atualize o navegador para ver as mudanças (F5)

EDITAR CONTEÚDO (HTML):
  → Modifique templates/index.html
  → Reinicie o servidor para ver as mudanças

EDITAR LÓGICA (JavaScript):
  → Modifique static/js/script.js
  → Atualize o navegador (F5)

EDITAR DADOS DO BACKEND:
  → Modifique os dicionários em app.py (PROJECTS, SKILLS, EXPERIENCES)
  → Acesse a API em http://localhost:5000/api/projects, etc

🚨 TROUBLESHOOTING
────────────────────

ERRO: "ModuleNotFoundError: No module named 'flask'"
  → Solução: pip install -r requirements.txt

ERRO: "404 Not Found for static file"
  → Verifique se o arquivo existe em static/css/ ou static/js/
  → Reinicie o servidor

PÁGINA NÃO CARREGA ESTILOS:
  → Limpe cache do navegador (Ctrl+Shift+Delete)
  → Recarregue a página (F5)
  → Verifique se app.py está rodando
